# Crime & Safety Dashboard

## Overview

The **Crime & Safety Dashboard** is an interactive data visualization project built using **Streamlit**. This dashboard provides insights into crime patterns, victim demographics, city-level crime distribution, and time-based crime trends. The goal of this project is to help users quickly understand crime patterns and identify areas of concern through visual analytics.

---

## Features

- Crime Type Distribution  
- Victim Gender Analysis  
- Victim Race Analysis  
- Crime Trend Over Time  
- Top Crime Cities  
- KPI Metrics (Total Cases, Most Common Crime, Cities Covered, Avg Victim Age)  
- Interactive Dashboard Layout  

---

## Dashboard Insights

The dashboard helps answer key questions such as:

- Which crime types occur most frequently?  
- Which cities experience the highest crime rates?  
- How does crime change over time?  

### Key Findings

- Domestic violence and burglary appear as the most common crime types  
- Houston and Dallas show higher crime counts  
- Crime trends fluctuate across time, indicating periodic patterns

## How to Run the Project

### 1. Clone the repository

```bash
git clone https://github.com/your-username/crime-dashboard.git
cd crime-dashboard
```

### 2. Install dependencies

```bash
pip install streamlit pandas plotly
```

### 3. Run the dashboard

```bash
streamlit run crime_dashboard.py
```
